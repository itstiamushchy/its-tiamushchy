Структура й відповіді (❌ помилки):
  • наявність аркушів Мета/Питання; ключі Мета; стовпці Питання;
  • порожнє питання; Тип поза {single, multiple, truefalse};
  • Правильна_відповідь: порожня / поза А–Г / вказує на порожній варіант / усі варіанти правильні;
  • Питань більше за кількість рядків.
Якість (⚠️ попередження):
  • баланс правильних А/Б/В/Г (перекіс до однієї літери / усі однакові);
  • дублікати питань; однакові варіанти в питанні; менше 2 варіантів;
  • single/multiple/truefalse-неузгодженості; надто довгі тексти; подвійні пробіли; «крякозябри»;
  • достатність питань у групах добору (Г1/Г2/…) для «Питань».
Manifest і покриття: дублі id/code; файли поза manifest; записи без файлу; id ↔ ім'я файлу; статистика по предметах/класах/типах.
